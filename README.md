# Break Reminder

A small Windows break reminder app with configurable work and rest periods.

## Features

- Configurable work time and rest time, minimum 1 minute.
- Starts work mode automatically when the app opens.
- Floating timer window that can be dragged and remembered between launches.
- Optional Windows auto-start.
- Single-instance protection, so opening the app twice will not create two timers.
- Tray icon menu for opening settings and controlling the timer.
- Start, pause, and end timer controls.
- Pause/end and early break exit require either the configured password or today's date in `mmdd` format.
- Optional forced break mode with fullscreen rest screen and limited keyboard blocking.
- Rest screen uses `assets/E.png` as a fullscreen distance/focus chart.
- English UI and English voice prompts.

## Run From Source

Install Python 3.11 or newer, then run:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python rest_reminder.py
```

## Build EXE

```powershell
.\build.ps1
```

The executable will be created at:

```text
dist\Break Reminder.exe
```
