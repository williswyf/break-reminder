# Microsoft Store Listing Draft

## Product Name

Eye Break Timer

## Short Description

A simple Windows break reminder that helps you rest your eyes and step away from the screen on schedule.

## Description

Eye Break Timer is a lightweight Windows break reminder for people who spend long periods at the computer. Set your work time and rest time, then let the app remind you when it is time to take a break.

The app includes a small floating timer, system tray controls, optional Windows auto-start, and a fullscreen rest view. During rest time, it can show a focus chart and optionally require a password to end the break early.

All settings are stored locally on your device. The app does not collect personal data, use analytics, or require an internet connection for its reminder features.

## Features

- Custom work and rest durations
- Floating countdown timer
- System tray menu
- Optional Windows auto-start
- Fullscreen rest screen
- Optional forced break mode
- Password-protected pause, end, and early break exit
- Local-only settings
